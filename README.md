# Promise

Promise implementation conforming the [Promises/A+](https://promisesaplus.com/) for browsers.

## Installing

~~~
    $ npm install @lvchengbin/promise --save
~~~

## Usage

### NodeJS

~~~javascript
    const Promise = require( '@lvchengbin/promise' );
~~~

### ES6

~~~javascript
    import Promise from '@lvchengbin/promise';
~~~
